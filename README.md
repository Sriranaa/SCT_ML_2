# Customer Segmentation using K-Means Clustering

## Objective

Group customers of a retail store into different segments based on their annual income and spending score using the K-Means Clustering algorithm.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Dataset

Mall_Customers.csv

## Steps Performed

1. Loaded and explored the dataset.
2. Checked for missing values.
3. Selected Annual Income and Spending Score as clustering features.
4. Used the Elbow Method to determine the optimal number of clusters.
5. Trained the K-Means clustering model.
6. Assigned customers to clusters.
7. Visualized customer segments and centroids.
8. Counted customers in each cluster.

## Conclusion

Successfully implemented K-Means Clustering to identify distinct customer groups based on purchasing behavior.
